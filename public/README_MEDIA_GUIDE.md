# オープニングメディア配置ガイド (AMU GACHA)

オープニング画面の「画像」と「動画」は、この `public/` フォルダに設置してください。

## 📁 設置場所とファイル名

### 1. オープニング画像（最初に出る静止画）
- **ファイルパス**: `public/cover.jpg` （または `public/cover.png`）
- **用途**: アプリ起動時の「タップしてスタート 👆」画面の背景として表示されます。

### 2. オープニング動画（タップ後に再生される動画）
- **ファイルパス**: `public/cover-video.mp4` （または `public/cover-video.webm`）
- **用途**: 画像をタップした直後に自動再生されるオープニング動画です。動画再生中に画面をタップするとメイン画面へスキップできます。

---

### 💡 ファイル追加手順
1. 左側のファイルエクスプローラーで `public` フォルダを右クリックし、「ファイルをアップロード」を選択します。
2. 上記のファイル名（`cover.jpg` および `cover-video.mp4`）に変更してアップロードしてください。
